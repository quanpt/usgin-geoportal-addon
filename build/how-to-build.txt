Pre-requisites:
	1. Tomcat 6.x installed with manager app
	2. Apache Ant installed and functional
	3. PostgreSQL database engine setup and running
	4. Database created and populated with Geoportal tables

1. Copy local.properties-example to local.properties
2. Adjust the configuration options in local.properties
3. Download the appropriate PostgreSQL JDBC driver for your installation: http://jdbc.postgresql.org/download.html
4. Run the Ant build.xml with the appropriate task
	a. clean: empty the build directory
	b. local.package: Compile the source code and generate the application in the build directory. For a local disribution
	c. remote.package: Same as above, but for a remote distribution
	d. war: Bundle whatever is in the build directory into a .war file
	e. local.deploy: Build the .war and deploy it locally
	f. remote.deploy: Build the .war and deploy it remotely
